# 📊 Dashboard de Análise de Vendas

Projeto de Business Intelligence desenvolvido em Python com foco em análise de desempenho comercial, utilizando Streamlit e Plotly para visualização interativa.

---

## 🚀 Objetivo

Transformar dados brutos de vendas em insights estratégicos para apoiar decisões de negócio, permitindo análise de faturamento, performance de produtos, regiões e status operacional.

---

## 📊 Principais Indicadores

- Faturamento total
- Volume de vendas
- Distribuição por produto
- Performance por região
- Status dos pedidos

---

## 🧠 Insights de Negócio

- Regiões com maior concentração de receita
- Produtos com maior contribuição para faturamento
- Eficiência operacional baseada no status dos pedidos
- Comportamento de vendas por categoria de produto
- Visão geral de performance do período analisado

---

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas
- Streamlit
- Plotly

---

## ⚙️ Pipeline de Dados

1. Ingestão de dados (CSV)
2. Limpeza e padronização
3. Transformação e tipagem
4. Análise exploratória
5. Visualização interativa
